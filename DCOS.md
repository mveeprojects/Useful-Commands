# DCOS

### List clusters
```shell
dcos cluster list
```

### Establish a connection with a new cluster
```shell
dcos cluster setup <ip_address>
```

### Rename a cluster
```shell
dcos cluster rename <current_cluster_name> <new_cluster_name>
```

### Attach to a cluster
```shell
dcos cluster attach <cluster_name>
```

### Login to a cluster
```shell
dcos auth login
```
